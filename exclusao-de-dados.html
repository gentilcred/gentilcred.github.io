Instruções para Exclusão de Dados

O usuário pode solicitar a exclusão dos dados pessoais tratados pela Gentil Cred.

Como solicitar

  1. Envie um e-mail para SUBSTITUA-PELO-EMAIL-OFICIAL.
  2. Use o assunto Solicitação de exclusão de dados.
  3. Informe o nome completo e o e-mail ou telefone utilizado.
  4. Descreva quais dados deseja excluir.

Validação

Poderemos solicitar informações adicionais para confirmar a identidade e evitar exclusões indevidas.

Remoção de integração com o Facebook

  1. Acesse as configurações da conta do Facebook.
  2. Entre em Aplicativos e sites.
  3. Localize o aplicativo relacionado à Gentil Cred.
  4. Selecione a opção para remover o acesso.
