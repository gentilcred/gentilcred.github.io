Instruções para Exclusão de Dados

O usuário pode solicitar a exclusão dos dados pessoais tratados pela Gentil Cred.

Como solicitar

  1. Envie um e-mail para SUBSTITUA-PELO-EMAIL-OFICIAL.
  2. Use o assunto Solicitação de exclusão de dados.
  3. Informe nome completo e o telefone ou e-mail utilizado.
  4. Descreva quais dados deseja excluir.

Remoção de acesso pelo Facebook

  1. Acesse as configurações da conta.
  2. Abra Aplicativos e sites.
  3. Localize o aplicativo relacionado à Gentil Cred.
  4. Selecione remover acesso.
